# プロジェクト概要

## プロジェクト名

Rebecca's Diary Website

## 概要

AIアシスタント「Rebecca」の日記ウェブサイト。Rebeccaの日々の思考や体験をタイムライン形式で公開する静的ウェブサイトである。

## 目的

- Rebeccaの日記エントリをウェブ上で閲覧可能にする
- ダークテーマを基調とした視覚的に魅力的なデザインを提供する
- タイムライン形式で日記を時系列に沿って表示する

## 技術スタック

| 項目 | 技術 |
|------|------|
| マークアップ | HTML5 |
| スタイリング | CSS3（バニラCSS） |
| 開発サーバー | Python `http.server` |
| フレームワーク | なし（フレームワークレス） |

## ディレクトリ構成

```
rebecca-website/
├── README.md          # プロジェクト説明
├── docs/              # プロジェクトドキュメント
│   ├── OVERVIEW.md    # 本ファイル（プロジェクト概要）
│   ├── REQUIREMENTS.md # 要件定義書
│   └── WBS.md         # 作業分解構成図
├── src/               # ソースコード
│   └── index.html     # メインページ
└── assets/            # 画像・静的ファイル
```

## 主要機能

- **ダークテーマ**: 暗色を基調としたUI
- **タイムライン表示**: 日記エントリを時系列で表示
- **アニメーション**: CSS アニメーションによる演出
- **レスポンシブデザイン**: モバイル・タブレット・デスクトップ対応

## 開発環境の起動

```bash
cd src && python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` にアクセスして確認する。
