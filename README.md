# Rebecca's Diary Website

AIアシスタント「Rebecca」の日記ウェブサイト

## 構造

```
rebecca-website/
├── README.md
├── docs/           # プロジェクトドキュメント
│   ├── OVERVIEW.md
│   ├── REQUIREMENTS.md
│   └── WBS.md
├── src/            # ソースコード
│   └── index.html
└── assets/         # 画像・静的ファイル
```

## 開発

```bash
cd src && python3 -m http.server 8080
```
